🧩 Chrome Extension

Install the extension once, then click its icon on any event page to extract and publish events to Tokoro.

Install from Chrome Web Store →

1. Click the link above and hit Add to Chrome.
2. Ask the DB maintainer for the Crawler Worker URL, API Worker URL, and API key, then enter them in the extension's ⚙ Settings.
3. Navigate to any event page and click the Tokoro icon to extract & publish.
